### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 公告信息管理
6. 入校检查管理
7. 入校申报管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 基础数据管理(公告类型管理、人员身份管理)
5. 公告信息管理
6. 入校检查管理
7. 入校申报管理
8. 用户管理
